<h1>Simple Web Application Written in HTML and Go</h1>
<p>The application uses go in the backend with <b>gorilla/mux</b> being used for routing, <b>html/template package</b> to display the data and structs to store the data.</p>

<img src="https://github.com/user-attachments/assets/a26a41d4-ff61-414f-a784-3feb067caf6f" alt="Application">

<h2>System Requirements</h2>
<p>Go must be installed in order to run the application.</p>

<h2>Build And Run</h2>
<p>Clone the github repository then get the required dependency.</p>
<pre>
  <code>
    git clone https://github.com/WDaily/html-go
  </code>
</pre>
<pre>
  <code>
    go get -u github.com/gorilla/mux
  </code>
</pre>

<p>Run the main.go file in a terminal.</p>
<pre>
  <code>
    go run main.go
  </code>
</pre>

<p>Search <b>http://localhost:8080/</b> on a browser to access the application.</p>
